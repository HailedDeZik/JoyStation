## 📝 Описание

**JoyStation** — это дипломный проект, представляющий собой полнофункциональный интернет-магазин видеоигр. 

**Основные возможности:**
- 🛍️ Просмотр каталога игр с фильтрацией
- 👤 Регистрация и аутентификация пользователей
- 🛒 Корзина покупок
- 💳 Оформление заказов
- 👨‍💼 Административная панель для управления товарами
- 🔐 JWT-токены для защиты API

---

## 🛠️ Стек технологий

**Backend:**
- Node.js
- Express.js
- SQLite
- JWT (JSON Web Tokens)

**Frontend:**
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📦 Требования

- Node.js (версия 14 и выше)
- npm или yarn
- Современный браузер

## ⚙️ Установка

1. Клонируйте репозиторий или скачайте файлы проекта
2. Откройте терминал в папке проекта
3. Установите зависимости:

```bash
npm install
```

## 🚀 Запуск

### Запуск сервера

```bash
npm start
```

Сервер запустится на **http://localhost:3000**

### Открыть фронтенд

Откройте файл `html/index.html` в браузере или используйте Live Server.

## 🔌 API документация

### Товары

**Получить все товары**
```
GET /api/products
```
Возвращает список всех доступных игр.

**Получить товар по ID**
```
GET /api/products/:id
```

**Добавить товар** (только админ)
```
POST /api/products
Content-Type: application/json

{
  "name": "Game Name",
  "price": 1999,
  "description": "Description"
}
```

**Удалить товар** (только админ)
```
DELETE /api/products/:id
```

### Аутентификация

**Регистрация**
```
POST /api/auth/register
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "password123"
}
```

**Вход**
```
POST /api/auth/login
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "password123"
}
```
Возвращает JWT-токен для использования в защищенных запросах.

### Пользователи

**Получить список пользователей** (только админ)
```
GET /api/users
```

## 💡 Разработка

### Добавить новую зависимость:
```bash
npm install package-name
```

### Структура кода

- **server.js** — основной файл сервера с маршрутами
- **db.js** — инициализация БД и SQL-запросы
- **data.js** — изначальные данные для БД
- **script.js** — логика фронтенда, работа с API

### Советы по разработке

1. JWT-токен передается в заголовке `Authorization: Bearer <token>`
2. Все защищенные эндпоинты требуют валидный токен
3. БД автоматически создается при первом запуске сервера
4. Используйте консоль браузера для отладки фронтенда
